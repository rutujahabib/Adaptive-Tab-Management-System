# Adaptive User-Centric Tab Management System

## Description
This project helps manage browser tabs using Hybrid Scoring and NLP to improve performance and reduce tab overload.

## Features
- Tab grouping using NLP
- Hybrid scoring (LRU, LFU, Working Set)
- RAM optimization
- User dashboard
